# Repository Disabled for DMCA Compliance

This repository previously contained IPTV playlist aggregation and playback resources.

Those materials have been removed and the project has been decommissioned in response to a copyright complaint.

## Current status

- Playlist outputs removed
- Web playback pages disabled
- Stream proxy/extraction tooling disabled
- Automated playlist generation workflow disabled

If you are the rights holder or platform operator and need additional remediation details, please use GitHub’s standard support/legal channel.
